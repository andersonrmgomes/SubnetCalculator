# Calculadora de Sub-redes

A Calculadora de Sub-redes é uma ferramenta baseada na web projetada para ajudar os usuários a calcular e visualizar informações de sub-redes IPv4. Ela oferece uma interface amigável para inserir endereços IP e prefixos CIDR, exibindo informações detalhadas da rede, representações binárias e distribuições de sub-redes.

## Funcionalidades

- **Cálculo de Sub-redes IPv4**: Insira um endereço IP e um prefixo CIDR para calcular os detalhes da rede.
- **Representação Visual**: Exibe a distribuição de sub-redes, mapas binários e capacidade da rede.
- **Design Responsivo**: Funciona perfeitamente em diversos dispositivos.

## Estrutura do Projeto

```
SubnetCalculator/
├── index.html       # Arquivo HTML principal
├── css/
│   └── style.css    # Folha de estilos da aplicação
├── js/
│   └── script.js    # Lógica JavaScript da aplicação
```

## Como Usar

1. Abra o arquivo `index.html` em qualquer navegador moderno.
2. Insira um endereço IPv4 no campo de entrada.
3. Selecione um prefixo CIDR no menu suspenso.
4. Clique no botão "Calcular" para visualizar os resultados.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da aplicação.
- **CSS3**: Estilização e layout.
- **JavaScript**: Lógica para cálculos de sub-redes e atualizações dinâmicas.

## Como Executar

1. Clone o repositório ou baixe os arquivos do projeto.
2. Navegue até o diretório do projeto.
3. Abra o arquivo `index.html` no navegador de sua preferência.
